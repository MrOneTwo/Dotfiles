# My dotfiles

- .Xresources - that's for urxvt terminal (place it in _~_)
- rc.lua      - that's for awesome wm (place it in _~/.config/awesome/_)
- init.vim    - that's for nvim (place it in _~/.config/nvim/_)
- .vimrc      - that's for vim (place it in _~_)
